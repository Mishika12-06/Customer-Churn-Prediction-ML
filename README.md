# Customer Churn Prediction using Machine Learning
This project predicts whether a customer will leave (churn) or stay with the company using classification models.
# Problem Statement
Customer churn is a major business problem. Retaining existing customers is more cost-effective than acquiring new ones. This project aims to identify customers likely to churn.
# Technologies Used
- Python
- Pandas
- Scikit-learn
# Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Handling Categorical Variables
4. Feature Scaling
5. Train-Test Split
6. Model Training
7. Model Evaluation
# Handling Imbalanced Data
The dataset was imbalanced, so SMOTE (Synthetic Minority Oversampling Technique) was applied to balance the classes before model training.
# Model Used
Random Forest 
# Model Performance
- Train-Test Split: 80-20
- ROC_AUC SCORE: 0.8271209432548602
- Evaluated on test dataset
# Business Insight
Identifying high-risk customers helps businesses take preventive actions and improve retention rates.
